代表结构作品

robomaster无人机


涉及内容

机械设计,材料力学,三维建模

项目简介

设计一台大载重室内定位四轴无人机,带两轴云台,可以搭载大量的小弹丸,平稳飞行的同时可持续发射.

云台
  1. 运动轴
    • 结构:6020直驱yaw轴,3508丝杆传动pitch轴
    • 实现效果:优势在于yaw轴响应快速而pitch轴有效节省重量同时利用丝杆的自锁性增强pitch的稳定性,在连续发射产生的后坐力的情况下实现稳定,且不用设计重力补偿.
  2. 发射机构
    • 结构:3508摩擦轮电机发射,枪管创新性使用3点定心+扳机结构
    • 实现效果:弹道更稳定,在6m距离下维持在100x100mm的发射散步,且扳机带来稳定的单发火控.
  3. 拨盘结构
    • 结构:八爪双层的渐开线拨盘结构
    • 实现效果:在理想状态下完全实现顺滑拨弹,且弹频最高可达23HZ
  4. 供弹链路
    • 结构:鹅颈供弹链路
    • 实现效果:缩短链路长度,减少弹道阻力,提高弹频.

机架设计

  1. 桨保
    结构:全包围桨保
    实现效果:稳定,重量轻
  2. 机臂
    结构:整体采用桨夹和中心板夹层的形式,配合桨保引入超静定结构,更加稳定,振动更小

总结:

在本科期间深度参与了很多赛事,产出了许多作品,在实践中不断查漏补缺,有良好的迭代思维,工程实践能力强.

仿真代表项目

3d打印航模


涉及内容

xflr5气动仿真,flow simulation,sw曲面建模

项目简介

一台双发前拉的常规布局平垂尾航模,使用fdm3d打印制作整体,巡航速度约为13m/s(仿真结果),使用profili进行翼型选型,Xflr5进行翼型仿真,sw flow simulation进行整机仿真,通过这些确定了我的目标载荷,再进一步进行设计.

项目成果

完整体验了航模的设计流程,从确定目标载荷与翼展限制,到选型翼型,再到仿真,整体建模,打印,再测试风洞,最后试飞的全过程.

总结

仿真是对物理世界的模拟,可以减少实验迭代成本,帮助项目推进,无论是气动还是有限元仿真都在不断地实践与创新中收获更多.
